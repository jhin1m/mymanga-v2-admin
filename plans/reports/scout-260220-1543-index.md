## Key Patterns to Remember

### Component State (Signals)
- 5 core signals: items, loading, total, pageIndex, pageSize
- Updated via .set() method
- Used in templates via signal() syntax

### Service Pattern
- Inject HttpClient + apiBase URL
- get{Entities}(params) — list with filters
- create{Entity}(payload) — POST
- update{Entity}(id, payload) — PUT
- delete{Entity}(id) — DELETE

### Modal Pattern
- Private showXxxModal(title, defaults, onSubmit) method
- Form component injects NZ_MODAL_DATA
- Validation in parent nzOnOk callback
- Promise resolves true/false for close/stay

### Template Pattern
- nzCard wrapper with search form
- Table with @for control flow
- Two pagination bars (top + bottom)
- Action buttons: Edit + Delete with popconfirm
